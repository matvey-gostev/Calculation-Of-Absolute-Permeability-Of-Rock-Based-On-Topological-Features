# Применение методов топологического анализа данных (TDA) и трансформеров для оценки абсолютной проницаемости разных пород по трехмерным данным симуляции структуры материи

**Выполняют студенты третьего курса:** Вотинова Анна, Комаров Сергей, Делов Никита, Гостев Матвей

---

## Мотивация

Абсолютная проницаемость горных пород является ключевым параметром в нефтегазовой геологии, гидрогеологии и проектировании подземных хранилищ. Традиционные методы ее определения трудозатратны, дорогостоящи и зачастую недостаточно точны для сложных гетерогенных структур.

Современные подходы к моделированию используют трехмерные изображения порового пространства, полученные методами микротомографии или физического симуляционного моделирования. Однако эффективное извлечение признаков из таких сложных структур остается нетривиальной задачей. **Топологический анализ данных** позволяет описывать форму данных инвариантным относительно деформаций способом, что делает его перспективным инструментом для характеризации пористых сред.

---

## Цели и задачи проекта

### Цель

Разработать подход к предсказанию абсолютной проницаемости горных пород, основанный на извлечении топологических признаков из трехмерных данных о структуре материи с последующим использованием архитектуры трансформеров.

### Задачи

1. **Анализ датасета:** изучить предоставленный набор данных, содержащий трехмерные представления структуры породы, полученные в результате физической симуляции.
2. **Предобработка данных:** реализовать первичную очистку, нормализацию и подготовку данных для топологического анализа.
3. **Пространственная фильтрация:** разработать и применить алгоритмы пространственной фильтрации для выделения значимых структурных элементов (пор, каналов, перетяжек).
4. **Извлечение топологических признаков:** используя методы TDA (персистентная гомология), вычислить топологические дескрипторы (например, диаграммы персистентности) для полученных структур. Сформировать эмбеддинги на их основе.
5. **Построение модели:** разработать архитектуру на основе трансформеров для обработки полученных эмбеддингов и регрессионного предсказания значения проницаемости.
6. **Валидация:** оценить точность предсказаний, сравнить с базовыми подходами (например, классические CNN или методы, основанные на пористости/извилистости).

## Данные

Используется открытый датасет, опубликованный в журнале *Scientific Data* (Nature):  
[**Three-dimensional representations of material structure obtained via physical simulation**](https://www.nature.com/articles/s41597-022-01664-0)

**Характеристики данных:**

- **Тип данных:** трехмерные массивы, представляющие распределение твердой фазы и порового пространства.
- **Происхождение:** данные получены в результате физической симуляции процессов формирования структуры материи.
- **Целевая переменная:** абсолютная проницаемость (скалярное значение) для каждого образца.

---

## Общая архитектура решения

1. **Входные данные:** 3D-изображение структуры породы.
2. **Предобработка:** пространственная фильтрация (например, медианная фильтрация, выделение связных компонент, бинаризация).
3. **Топологический анализ (TDA):**  
   - Построение фильтрации на основе значений (по расстоянию до твердой фазы или по уровню пористости).  
   - Вычисление персистентной гомологии (H0, H1, H2) для описания компонент связности, циклов и полостей.  
   - Векторизация диаграмм персистентности (например, с использованием persistence landscapes, persistence images).
4. **Моделирование:** трансформерная архитектура, принимающая на вход последовательность топологических признаков (или их комбинаций) и выдающая прогноз проницаемости.

---

## Ожидаемые результаты

- Пайплайн предобработки 3D-структур и извлечения топологических признаков с использованием TDA.
- Набор эмбеддингов, характеризующих топологические особенности порового пространства.
- Модель на основе трансформеров, обеспечивающая предсказание абсолютной проницаемости.
- Сравнительный анализ эффективности предложенного подхода по сравнению с традиционными методами (пористость-проницаемость, CNN).
